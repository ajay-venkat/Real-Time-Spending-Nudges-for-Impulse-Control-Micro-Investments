### Prerequisites
- **Node.js** 18+ and **npm**
- **Google Gemini API Key** (for AI nudges) â€” [Get one here](https://aistudio.google.com/app/apikey)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/ajay-venkat/Real-Time-Spending-Nudges-for-Impulse-Control-Micro-Investments.git
cd Real-Time-Spending-Nudges-for-Impulse-Control-Micro-Investments

# 2. Install dependencies
npm install

# 3. Set up environment variables
```

Create a `.env.local` file:
```env
GOOGLE_GENAI_API_KEY=your_gemini_api_key
```

> ğŸ’¡ **No database setup needed!** The app uses browser `localStorage` for all data persistence. Mock data auto-seeds on first visit.

### Running the App

```bash
# Start the development server (runs on port 9002)
npm run dev

# Start the Genkit AI Dev UI (optional, for testing AI flows)
npm run genkit:dev
```
